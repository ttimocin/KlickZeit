# ZeitLog Wear OS Uygulaması

Samsung saatler için ZeitLog uygulaması.

## Özellikler

- Güncel saat gösterimi
- Bugünkü giriş saatini gösterir
- Firebase ile telefon uygulamasıyla senkronize

## Kurulum

1. Android Studio'da projeyi açın
2. `google-services.json` dosyasını `wear/app/` klasörüne ekleyin
3. Wear OS emülatörü veya gerçek saat bağlayın
4. Run butonuna tıklayın

## Gereksinimler

- Android Studio
- Wear OS 2.0+
- Firebase hesabı (telefon uygulamasıyla aynı)
