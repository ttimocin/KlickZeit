Política de Privacidade

Última atualização: 20 de junho de 2026

1. Introdução

KlickZeit é um aplicativo gratuito de controle de tempo de trabalho da TayTek. Esta política explica quais dados são coletados e como são processados.

2. Dados coletados

• E-mail (login por e-mail, Google ou Apple) • ID de sessão convidado anônimo • Horários de entrada/saída, durações, pausas e férias • Configurações do app • Pontuações, melhores tempos e apelidos opcionais • Código de usuário apenas para convidados (ex. KZ-ABCD1234) • Registros locais no dispositivo

3. Uso

Dados usados para funcionalidades, backup na nuvem opcional, jogos/rankings e verificação de atualizações. Dados de trabalho não são vendidos.

4. Publicidade

Banners podem ser exibidos na tela inicial via Google AdMob (Google LLC). ID de publicidade pode ser usado. Política Google: https://policies.google.com/privacy. Dados de trabalho não são usados para perfis publicitários.

5. Armazenamento

• Local (AsyncStorage) • Backup Firebase opcional • E-mail no perfil para usuários logados; código só para convidados • Rankings visíveis a outros usuários • Dados de trabalho vinculados à sua conta

6. Segurança

Firebase Authentication, regras Firestore e transmissão criptografada (HTTPS/TLS).

7. Exclusão

Configurações > Excluir conta para remoção permanente. Sair separadamente disponível. Desinstalar remove dados locais.

8. Terceiros

Firebase (Google), Google AdMob (banners), Google Sign-In, Sign in with Apple.
