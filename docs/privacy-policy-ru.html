Политика конфиденциальности

Последнее обновление: 19 июня 2026 г.

1. Введение

KlickZeit — бесплатное приложение учёта рабочего времени от TayTek. Эта политика описывает собираемые данные и их обработку.

2. Собираемые данные

• Электронная почта (вход по email, Google или Apple) • Анонимный ID гостевой сессии • Время прихода/ухода, перерывы и отпуска • Настройки приложения • Игровые результаты и псевдонимы • Код пользователя только для гостей (напр. KZ-ABCD1234) • Локальные записи

3. Использование

Данные используются только для функций приложения, облачного резервирования, игр/рейтингов и проверки обновлений. Без рекламы и продажи третьим лицам.

4. Хранение

• Локально (AsyncStorage) • Опционально Firebase Firestore • Email для авторизованных; код только для гостей • Рейтинги видны другим пользователям

5. Безопасность

Firebase Authentication, правила Firestore, шифрованная передача (HTTPS/TLS).

6. Удаление

Настройки > Удалить аккаунт — полное удаление. Выход отдельно. Удаление приложения стирает локальные данные.

7. Сторонние сервисы

Firebase (Google), Google Sign-In, Sign in with Apple.
